Solus MS
A browser-based game launcher with a Steam-like UI. Hosts Hollow Knight, Slope, Minecraft 1.12.2 and more — all opening fullscreen in a new tab. Features tab cloaking, reflection cloaking, invisibility cloaking, panic key, and about:blank URL spoofing. Written entirely in HTML.

Spooftify V4
Advanced offline music player with a clean Spotify-inspired UX. Import and access high-quality music files locally — no streaming, no account. Desktop version (v4.2.1) is a downloadable single-file HTML app; the mobile edition runs directly in-browser.

ZenLit o2
A fast, minimalist chess engine GUI and wrapper built in JavaScript. Runs a highly optimised WebAssembly (WASM) engine achieving ~3650 ELO with under 5000ms movetime and 98.4% accuracy. Zero dependencies, fully browser-native.
HTML / WebAssembly
Calculator 2
A comprehensive "Swiss Army Knife" calculator. Covers standard & scientific modes, symbolic algebra/CAS, calculus (derivatives, integration, ODEs), linear algebra (matrices, vectors), statistics, regression, QR-code graphing, complex numbers, base conversion, unit/currency conversion, and offline support via service worker.

Source-accurate web port of the popular Steam game by Mr Suicide Sheep (10k+ reviews, 5 stars). The game was built with Construct and exported as HTML/JS — this port extracts the package.nw bundle and runs it natively in-browser with zero modifications to gameplay or assets.
